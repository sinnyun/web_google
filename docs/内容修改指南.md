# 网站内容修改指南

这份指南帮助不懂代码的人修改网站内容。所有需要修改的文件都已添加详细的中文注释。

## 🎯 常见修改任务

### 1. 修改项目内容

**文件：`data.ts`**

#### 修改项目标题
```typescript
title: 'Neon Void',  // 改为：title: '你的项目名',
```

#### 修改项目描述
```typescript
description: 'An exploration of...',  // 改为你的项目描述
```

#### 修改项目分类
```typescript
category: 'Visual Identity',  // 改为：'平面设计' 或其他分类
```

#### 修改主页展示的项目
```typescript
export const HOME_SHOWCASE_ORDER = [3, 5, 9, 6, 1];
// 数字对应项目的 numericId
// 调整顺序或增删数字来改变主页展示
```

#### 修改项目详情页布局
```typescript
layout: [1, 2, 3, 2, 1],
// 1 = 单列（大图）
// 2 = 两列并排
// 3 = 三列并排
// 按顺序排列图片布局
```

## 🖼️ 添加新项目

### 步骤1：准备图片

1. 在 `public/images/` 文件夹下创建新文件夹
2. 文件夹名使用英文小写，用连字符分隔（如 `my-new-project`）
3. 放入图片：
   - `cover.jpg` - 封面图
   - `detail-1.jpg`, `detail-2.jpg`, ... - 详情图

### 步骤2：添加项目数据

在 `data.ts` 的 `PROJECTS` 数组中添加：

```typescript
{
  id: 'my-new-project',        // 与文件夹名相同
  numericId: 10,               // 下一个数字
  layout: [1, 2, 3],           // 设计布局
  title: '我的新项目',
  category: '项目分类',
  year: '2024',
  location: '北京, 中国',
  description: '项目描述...',
  coverImage: '/images/my-new-project/cover.jpg',
  images: getLocalImages('my-new-project', 10),  // 10是图片数量
  tags: ['标签1', '标签2', '标签3'],
},
```

### 步骤3：（可选）添加到主页

在 `HOME_SHOWCASE_ORDER` 中添加项目的 `numericId`：

```typescript
export const HOME_SHOWCASE_ORDER = [3, 5, 9, 6, 1, 10];  // 添加10
```

## ⚠️ 注意事项

1. **保存文件**：修改后务必保存文件
2. **刷新浏览器**：保存后刷新浏览器查看效果
3. **保持格式**：
   - 不要删除引号 `"` 或 `'`
   - 不要删除逗号 `,`
   - 不要删除标签的开始和结束部分
4. **备份**：修改前建议备份原文件
5. **测试**：修改后测试所有页面是否正常

## 📝 文件说明

### 核心文件
- **data.ts** - 所有项目数据，最常修改
- **index.html** - 全局样式和颜色配置
- **App.tsx** - 导航栏和路由配置
- **AboutOverlay.tsx** - 关于页面的个人信息

### 页面文件（一般不需要修改）
- **Home.tsx** - 主页幻灯片
- **Work.tsx** - 项目列表页
- **ProjectDetail.tsx** - 项目详情页

### 组件文件（一般不需要修改）
- **Cursor.tsx** - 自定义光标
