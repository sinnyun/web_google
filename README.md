# Sinnyun Portfolio - 个人作品集网站

**注意：此网站由AI生成，我不懂代码也不懂英文。所有代码文件都有详细的中文注释说明。**

一个基于 React + TypeScript 构建的现代化个人作品集网站，专为设计师、摄影师和创意工作者打造。网站包含精美的加载动画、自定义光标、流畅的页面切换效果，以及灵活的作品展示系统。适合用来展示个人作品、摄影作品、设计项目等。

## ✨ 核心亮点

- 🎨 **开箱即用**: 包含9个完整的示例项目，可以直接使用或替换成自己的作品
- 📝 **全中文注释**: 所有代码文件都有详细的中文说明，不懂代码也能看懂
- 🎯 **简单修改**: 只需要改文字和图片路径，不需要写代码
- 🚀 **专业级设计**: 精美的视觉效果和流畅的动画，展示您的作品更有品质感
- 🛠️ **易于维护**: 修改配置文件即可更新内容，刷新浏览器立即看到效果

## 🚀 快速开始

### 前置条件

需要先安装 **Node.js**（推荐 v16 或更高版本）

> 💡 **什么是Node.js？** 它是一个让网站能在本地电脑运行的工具。就像你需要安装Office才能打开Word文档一样，运行这个网站需要先安装Node.js。
> 
> 下载地址：访问 [nodejs.org](https://nodejs.org) 下载安装包，选择"LTS"版本（长期支持版）。

### 运行步骤

1. **克隆或下载项目到本地**
   - 如果懂Git：使用 `git clone` 命令
   - 如果不懂Git：直接下载ZIP压缩包并解压

2. **安装依赖包**
   ```bash
   npm install
   ```
   > 💡 这个命令会自动下载项目需要的所有工具包，需要等待几分钟。只需要运行一次。

3. **启动开发服务器**（在电脑上预览网站）
   ```bash
   npm run dev
   ```
   > 💡 运行后会显示一个网址（如 http://localhost:5173），在浏览器打开这个网址就能看到网站。
   > 
   > 📝 开发模式下，修改代码保存后网站会自动刷新显示最新效果。

4. **构建生产版本**（准备上线部署）
   ```bash
   npm run build
   ```
   > 💡 这个命令会生成优化后的网站文件，放在 `dist` 文件夹中，用于正式部署到服务器。

5. **预览生产版本**
   ```bash
   npm run preview
   ```
   > 💡 在本地预览构建后的生产版本，确保没有问题再部署。

## 🎨 自定义样式

项目使用 Tailwind CSS 框架，所有样式都通过"类名"来控制。

### 常用样式类名说明

如果你想调整某些元素的样式，可以修改代码中的类名：

- **间距调整**:
  - `p-4`: 内边距（元素内部的空白）
  - `m-4`: 外边距（元素外部的空白）
  - 数字越大，间距越大（如 `p-2`, `p-4`, `p-8`）

- **颜色调整**:
  - `bg-stone-950`: 背景颜色（stone-950是深灰黑色）
  - `text-stone-50`: 文字颜色（stone-50是浅灰白色）
  - 可替换为其他颜色，如 `bg-black`（黑色）、`text-white`（白色）

- **布局方式**:
  - `flex`: 弹性布局（元素横向或纵向排列）
  - `grid`: 网格布局（元素按网格排列）
  - `block`: 块级布局（元素独占一行）

- **响应式设计**:
  - `md:flex`: 在中等屏幕（平板）及以上显示为flex布局
  - `lg:hidden`: 在大屏幕（电脑）上隐藏元素
  - `hidden md:block`: 手机端隐藏，平板及以上显示

> 💡 **提示**：如果不确定某个类名的作用，可以在 [Tailwind CSS 官方文档](https://tailwindcss.com) 搜索查看。

## 🌐 部署上线指南

将网站部署到互联网上，让所有人都能访问。

### 方式1：Vercel 部署（推荐，免费）

Vercel是一个免费的网站托管平台，最适合这类项目。

**步骤：**

1. **注册Vercel账号**
   - 访问 [vercel.com](https://vercel.com)
   - 使用GitHub账号登录（推荐）

2. **上传代码到GitHub**
   - 在GitHub创建一个新仓库（repository）
   - 将项目代码推送到GitHub仓库

3. **在Vercel导入项目**
   - 点击 "New Project"
   - 选择你的GitHub仓库
   - Vercel会自动检测到Vite项目

4. **配置构建设置**
   - **Framework Preset**: Vite
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
   - 点击 "Deploy" 开始部署

5. **完成**
   - 等待几分钟，Vercel会自动构建和部署
   - 获得一个免费的网址（如 `your-project.vercel.app`）
   - 每次推送代码到GitHub，网站会自动更新

> 💡 **优点**：完全免费、自动部署、提供HTTPS、速度快

### 方式2：Netlify 部署（免费）

**步骤：**

1. 在本地构建项目：`npm run build`
2. 访问 [netlify.com](https://netlify.com) 注册账号
3. 将 `dist` 文件夹拖拽到Netlify页面上传
4. 配置重定向规则以支持 React Router（Netlify会自动识别）
5. 获得免费网址（如 `your-site.netlify.app`）

### 方式3：自己的服务器

如果你有自己的服务器（如阿里云、腾讯云）：

1. 构建项目：`npm run build`
2. 将 `dist` 文件夹中的所有文件上传到服务器
3. 配置Web服务器（如Nginx或Apache）指向这些文件
4. 确保服务器支持单页应用的路由配置

> ⚠️ **注意**：项目使用Hash路由（`#/`），在大多数服务器上都能直接运行，无需特殊配置。

### 自定义域名（可选）

部署成功后，如果想使用自己的域名（如 `www.yourname.com`）：

1. 在域名注册商（如阿里云、GoDaddy）购买域名
2. 在Vercel或Netlify的设置中添加自定义域名
3. 按提示配置DNS解析
4. 等待DNS生效（通常几分钟到1小时）

> ✅ **推荐**：初学者建议使用Vercel或Netlify，它们提供免费的HTTPS和自动部署，非常方便。
